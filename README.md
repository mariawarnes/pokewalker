# Interactive Pokewalker

A Pen created on CodePen.io. Original URL: [https://codepen.io/mariawarnes/pen/wvYRPEr](https://codepen.io/mariawarnes/pen/wvYRPEr).

Semi-interactive recreation of the Pokewalker by Nintendo. Press and hold the middle button and watch the screen switch on. Try pressing the walk button to simulate walking to see the step counter rise.
